
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ankit628792/TheNeuronClub.git
   ```
2. change directory to TheNeuronClub 
   ```sh
   cd TheNeuronClub
   ```   
3. Install NPM packages
   ```sh
   npm install
   ```
4. run the project
   ```sh
   npm run dev
   ```   
   
Preview: https://neuron-club.vercel.app/
